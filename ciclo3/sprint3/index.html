En las siguientes votaciones a realizar hay presente cuatro candidatos. Este aplicativo permite conocer las ideologías políticas de cada uno de ellos, presionando sobre la foto. Este app, en etapas posteriores permitirá realizar el proceso de votación de manera digital, de acuerdo a la cantidad de candidatos, en los cuales estará un botón que permitirá escoger el candidato predilecto de cada votante. Cada persona solo puede votar una única vez.

Consideraciones

Una vez el votante haya marcado y confirmado su voto, este no podrá ser cambiado. Esto con el fin de hacer más seguro el proceso y disminuir las probabilidades de riesgo, por ello, el ciudadano siempre deberá confirmar el voto, para que este sea válido.
